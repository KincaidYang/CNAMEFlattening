# CNAMEFlattening
## 相关说明
本脚本用以拉平 CNAME 记录，当前仅支持 DNSPod、华为云DNS。

DNSPod DNSPod DNS

HuaweiCloud 华为云DNS

请根据实际需要选择对应的脚本使用。

注意：本脚本仅测试了与腾讯云 CDN、腾讯云 EdgeOne、华为云 CDN、天翼云 CDN、阿里云 CDN 的兼容性，其他 CDN 厂商未测试兼容性，若有其他厂商需求请自行测试或提交Issue。
